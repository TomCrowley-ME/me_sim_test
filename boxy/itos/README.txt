===================================================================
ABOUT
===================================================================

This is the Moon Express master ITOS directory, which contains most
of the files needed to configure/run ITOS.



===================================================================
REQUIREMENTS
===================================================================

First of all, you need to install ITOS natively on your Linux system
or in a virtual machine. Take a look at the ITOS Setup Guide wiki
to get going with installation:

http://crushinator.moonexpress.com:3000/projects/moonex-software/wiki/ITOS_Setup

To autogenerate ITOS recfiles and pages from the MoonEx C&T Dictionary first
clone bender and make sure your BENDER_PATH environment variable is set correctly.
This is described in the C&T Wiki:

http://crushinator.moonexpress.com:3000/projects/moonex-software/wiki/Command_and_Telemetry_Dictionary



===================================================================
GETTING STARTED
===================================================================

Clean out any old odb/rec/page files:

  $ ./makeclean


Autoconfigure ITOS rec files and pages (for MTV-1 and Simulator):

  $ ./gen_itos_config.py MTV-1


Build the ITOS operational database:

  $ ./makeodb


Launch ITOS:

  $ itos




===================================================================
VIEWING TELEMETRY PAGES
===================================================================

At startup, the autogenerated master telemetry page is displayed to the user.
Each system contains a set of buttons that link to the autogenerated page for
the respective telemetry packet.

Example:
To see the sim_data_bus page, click the button under SIMULATOR labeled "sim_data_bus_msg".
