CSU TAF : Thruster Arm & Fire Requirements

Requirements

 

Requirement ID

TAF-L6-001

Title

Pointing Arm Thrusters & Duration

Req. Type

Functional

Parent

ACT-L5-001

Children

None

Requirement Text

The TAF shall arm all ACS thrusters for a no less than the software cycle time at every cycle, when in pointing mode and active control is enabled.

Rationale


Verification

Test (csu_taf_thruster_arm_fire_test.m)
